IT General Guide

Solutions for daily IT problems and technical support

Windows Image Setup

  • Use Rufus or Windows Media Creation Tool
  • USB drive at least 8 GB
  • Choose appropriate ISO (Windows 10/11 Pro)
  • GPT for UEFI / MBR for Legacy

  • Boot from USB (F12, ESC, Del)
  • Choose Custom: Install Windows only
  • Delete all partitions
  • Select Unallocated Space and press Next

  • Check Drivers
  • Run Windows Update
  • Install apps: Office, Company Portal, AnyDesk
  • Join Domain if required
Performance Issues
  • RAM: If 90%+, needs upgrade
  • HDD: Upgrade to SSD is fastest fix
  • CPU: Check what's consuming processor
  • Storage: Clear %temp% and empty recycle bin
  • Restart: Fixes 80% of problems
Task Manager

Ctrl + Shift + Esc

Performance Monitor

perfmon /res

Computer Not Working
Power Issues:
  • Check cable
  • Try another socket
  • Remove battery 30 seconds
  • Try different power adapter
Boot Issues:
  • Try external monitor
  • Listen for beep sounds
  • Try Safe Mode
  • Clean RAM modules
Headset Issues
  • Check if not muted
  • Verify in Playback devices
  • Try different USB port
  • Test on your phone
  • Update/Rollback driver
Audio Troubleshooter

Service Desk Guidelines

Standard operating procedures

Reported Incidents
  • Open HelpDesk folder
  • Select correct category and write clear description
  • Follow up daily on transferred incidents
  • Manager approval for hardware changes
Closed Incidents
  • Document full troubleshooting steps
  • If resolved without action: Solved with No Action Taken
  • Select appropriate transferred area
Escalation Rules
  • Window: Sun-Thu, 08:00-16:00
  • Set status to Awaiting Vendor
  • Include: serial, model, issue, dates, user details
Device Cycles
  • Cycle 1: Preparation - Obtain approvals
  • Cycle 2: Moving & Tracking - Label devices
  • Cycle 3: Receipt - Verify and sign-off
